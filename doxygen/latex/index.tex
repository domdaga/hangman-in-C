\hypertarget{index_autotoc_md0}{}\doxysection{\texorpdfstring{Hangman Game in C}{Hangman Game in C}}\label{index_autotoc_md0}
This is a simple implementation of the classic Hangman game in C. The game allows the user to guess a word letter by letter, with a limited number of attempts. The game also logs all actions and outputs to a file for later review.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Compilation}{Compilation}}\label{index_autotoc_md1}
starting the programm with run.\+sh will start with the word Katze, if you prefer another word read the README-\/file. you will build the programm with the build.\+sh file. This will compile the .c file using "{}gcc -\/std=c99 -\/\+Wall -\/\+Wextra -\/pedantic -\/\+Wno-\/unused-\/parameter"{} and automaticly execute the file in its normal mode.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Random Word Selection}}\+: The game can select a random word from a file containing a list of words.
\item {\bfseries{Logging}}\+: All game actions, including guesses, correct/incorrect attempts, and game state, are logged to a file ({\ttfamily game\+\_\+log.\+txt}).
\item {\bfseries{Hangman Visualization}}\+: The game displays the hangman\textquotesingle{}s gallows and updates it based on the number of remaining attempts.
\item {\bfseries{Word Reset}}\+: The game can reset the status of words in the word list file, allowing words to be reused in future games.
\item {\bfseries{Input Validation}}\+: The game ensures that only valid letters are entered as guesses.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{Requirements}{Requirements}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{C Compiler}}\+: A C compiler such as {\ttfamily gcc} is required to compile the code.
\item {\bfseries{Word File}}\+: A text file containing words to be used in the game. Each word should be prefixed with a status (0 or 1) and separated by a comma (e.\+g., {\ttfamily 0,apple}). A word file is given by us but you could also create your own.
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysubsection{\texorpdfstring{Run the Game}{Run the Game}}\label{index_autotoc_md4}
